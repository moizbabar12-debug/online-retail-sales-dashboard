# Online Retail Sales Dashboard

This project analyses an online retail dataset from a UK based gift wholesaler. The aim is to clean the data, explore sales patterns, and build an interactive dashboard that helps managers understand revenue, product performance, customer behaviour and country-level sales.

## Project Aim

The aim of this project is to turn raw retail transaction data into clear business insights using data cleaning, exploratory data analysis and dashboard visualisation.

## Main Work Completed

- Loaded and inspected the Online Retail dataset
- Checked missing values, duplicates and invalid records
- Removed duplicate rows
- Removed cancelled or invalid transactions with non-positive quantities or prices
- Converted invoice dates into proper datetime format
- Created a LineTotal column for revenue
- Checked outliers using the IQR method
- Built visualisations for revenue, products, countries and order values
- Created an interactive dashboard with filters for date period, country and minimum order value

## Dashboard Features

The dashboard includes:

- Revenue over time
- Top 10 products by revenue
- Revenue by country
- Order value distribution
- Customer orders vs total spend
- Correlation heatmap
